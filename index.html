MACBOOK TERMINAL

Alterar o delay para mostrar o Dock
defaults write com.apple.dock autohide-delay -float 0; killall Dock
Travar o tamanho do Dock
defaults write com.apple.Dock size-immutable -bool true; killall Dock
Voltar o Dock ao seu estado padrão
defaults delete com.apple.dock; killall Dock
Desligar em horário específico
sudo pmset schedule sleep "10/30/2024 21:00:00"
Testar DISCO SÓLIDO (macOS)
time dd if=/dev/zero of="/Volumes/NOMEDISCO/testfile" bs=100m count=1
Testar FlashDrive (macOS)
sync
time dd if=/Volumes/NOMEDOPENDRIVE/testfile of=/dev/null bs=100m
Como desativar e apagar backups locais (macOS)
Apagar todos os snapshots locais: (copia & editar) - isto força o sistema a tentar apagar o máximo possível
sudo tmutil thinlocalsnapshots / 999999999999
Desativar backup local (definitivamente): (copia & editar) — em versões recentes do macOS pode não surtir efeito
sudo tmutil disablelocal
Insira o comando para encontrar seu IP (macOS)
Para conexões a cabo: ipconfig getifaddr en1 | Wi-Fi: ipconfig getifaddr en0
ipconfig getifaddr $(route get default | grep interface | awk '{print $2}')

Terminal para otimizar computador (macOS)

# Desativar
setopt +o nomatch

# Limpar o cache do sistema
sudo rm -rfv /Library/Caches/*
sudo rm -rfv ~/Library/Caches/*
sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder
sudo sysctl vm.swappiness=10
sudo periodic daily weekly monthly

# Limpar logs do sistema
sudo rm -rf /private/var/log/*
sudo rm -rfv /var/log/*
sudo rm -rfv /Library/Logs/*
sudo rm -rf /Library/Logs/*
sudo rm -rfv ~/Library/Logs/*
sudo rm -rf ~/Library/Logs/*

# Limpar cache do Google Chrome (Mac)
rm -rf ~/Library/Caches/Google/Chrome

# Forçar a liberação de memória
sudo purge
sync; sudo purge

sudo mdutil -E
sudo rm -rf /.metadata_never_index
sudo mdutil -E /

# Limpar cache do Safari
rm -rf ~/Library/Caches/com.apple.Safari
rm -rf ~/Library/Caches/com.apple.Safari/*

# Verificar e reparar permissões de disco
sudo diskutil resetUserPermissions / `id -u`

# Atualizar os aplicativos do sistema 
sudo softwareupdate -ia
softwareupdate --install --all

# Ativar
setopt nomatch
*Aviso:* muitos desses comandos removem caches e logs — use com atenção. Alguns comandos podem exigir senha de administrador.
